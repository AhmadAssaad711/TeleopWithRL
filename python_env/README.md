# Python Environment Area

This folder groups the operational entry points and results for the active
`TeleopEnv`-based Python environment.

Use `scripts/` for standard runs and `results/` for generated outputs.

Standard agent reruns for the current environment are written under:

- `python_env/results/standard_agents_new_env`

The core implementation still lives at the package root:

- `teleop_env.py`
- `train_dqn.py`
- `train_q_learning.py`
- `mrac_controller.py`
- `benchmark_agents.py`
